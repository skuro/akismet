akismet by skuro

Clojure library to ask Akismet for blog comment spam detection

Akismet filters out your comment and track-back spam for you, so you can focus on more important things. It provides a lightweight HTTP API to call its services. `akismet` provides a Clojure API that allows for easy interaction with Akismet web services from your code. The main API includes: - `valid-key?` - `spam?` - `spam!` - `ham!`
